# Sales Performance Dashboard (Power BI)

## Project Overview
This project presents a comprehensive Sales Performance Dashboard built using Power BI. The goal of this analysis is to derive meaningful business insights from raw sales data and help stakeholders make data-driven decisions.

## Objectives
- Clean and organize raw sales data
- Analyze revenue trends over time
- Identify top-selling products
- Evaluate performance across regions and customer segments
- Create a professional and interactive dashboard
- Provide actionable business insights and recommendations

## Tools and Technologies Used
- Power BI
- Microsoft Excel (Financial Sample Dataset)
- Data Cleaning and Transformation (Power Query)

## Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Units Sold
- Profit Margin

### Visualizations
- Sales by Month
- Sales by Country
- Sales by Segment
- Top Products by Sales

## Key Insights
- Sales show seasonal trends with peak performance in specific months.
- A few countries contribute the majority of revenue.
- One customer segment dominates overall sales.
- Top products generate a large share of total revenue.
- Profitability is affected by discount strategies.

## Recommendations
- Focus marketing efforts during high-performing months
- Expand strategies in underperforming regions
- Target high-value customer segments
- Promote top-selling products strategically
- Optimize discount policies to improve profit margins
